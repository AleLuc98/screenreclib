int SRDecoder::decodePacket ( AVPacket *  packet)

The method wraps the avcodec_send_packet which supplies raw packet data as input to a decoder. Internally, this call will copy relevant AVCodecContext fields, which can influence decoding per-packet, and apply them when the packet is actually decoded.

Parameters
packetis the packet that needs to be decoded
Exceptions
SRNullPacketExceptionif the received packet is null
SRNullContextExceptionif the decoder has not been initialized correctly
