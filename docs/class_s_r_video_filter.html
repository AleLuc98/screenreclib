◆ enableBasic()

void SRVideoFilter::enableBasic ( )

This method adds the basicFilter to the filters chain. When the method is called, all the necessary modules and object to support the filter are instantiated. The filter is set up by using information both from the input codec context and both from the output codec context, to understand how the frame has to be processed.

Exceptions
InvalidFilterParametersExceptionif the decoder or the encoder are not correctly initialized
BufferAllocationExceptionif the malloc for allocation the video buffer fails

◆ enableCropper()

void SRVideoFilter::enableCropper ( )

This method adds the cropFilter to the filters chain. When the method is called, all the necessary modules and object to support the filter are instantiated. The filter is set up by using information from the input codec context (to understand the input resolution and pixel format) from the output codec context (which has to be set with the cropped resolution before calling this method) and from the crop settings to understand the crop offset.

Exceptions
SRFilterExceptionif the crop filter cannot be initialized

◆ filterFrame()

AVFrame * SRVideoFilter::filterFrame ( AVFrame *  input_frame)

The filterFrame() method is responsible for processing the frame according to the defined filters chain. The filters chain can be created by using the basic filter (responsible for preparing the decoded frame for the encoder) and the crop filter (responsible for cropping the frame). At least basicFilter has to be enabled. The input frame is deleted after the process and cannot be used anymore.

Parameters
input_frameis the raw frame received from the decoder
Exceptions
CropperExceptionif an error while enabling the video cropper occurs
Returns
the pointer to the processed frame
