◆ initConvertedSamples()

int SRAudioFilter::initConvertedSamples ( int  frame_size)

The method allocates as many pointers as there are audio channels. Each pointer will later point to the audio samples of the corresponding channels (although it may be NULL for interleaved formats).

It also allocates memory for the samples of all channels in one consecutive block for convenience.

Parameters
frame_sizeindicates the number of samples added to the buffer
